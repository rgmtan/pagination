pagination
==========
Pagination class built for dietcake use. 

How to install : 

1. Go to your dietcake repository
```
    cd myrepo/lib/
```
2. Clone repo
```
    git clone https://github.com/rtanklab/pagination.git
```
3. configure in bootstrap.php
```
    add : 
    // lib
    require_once LIB_DIR.'pagination/pagination.php';
```
4 . Use it and enjoy!
